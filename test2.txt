github와 연동하기

1. github에 저장소생성하기
2. 저장소 주소를 복사
3. 소스트리에서 clone버튼클릭
4. clone 화면의 원격 저장소 주소 입력란에 생성한 github 저장소 주소 붙여너기
5. 로컬 컴퓨터에 저장한 위치 선정하기
6. clone버튼 클릭하기

원격설정
1. 설정 버튼 클릭
2. 원격 저장소 편집
3. url 정보 중 https://github에서 발급한 토큰정보@github.com/본인아이디/프로젝트.git으로 수정
4. 확인버튼 클릭하여 설정 완료
5. 파일 수정후 커밋을 진행하고 push 버튼 클릭하여 업로드 확인
