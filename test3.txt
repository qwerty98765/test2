로컬 컴퓨터에서 먼저 git 저장소생성하고 나중에 서버에 등록
1. 소스트리에서 create버튼 클릭
2. git 저장소 위치 및 프로젝트명
3. 저장소 생성 후 버전 관리할 파일 생성
4. 커밋 눌러 버전 관리 시작
5. 깃허브 사이트에서 원격 저장소 생성(로컬 컴퓨터의 저장소 이름과 동일하게 생성)
6. 원격저장소 url 복사
7. 소스트리에서 설정 버튼 클릭
8. 원격지 주소추가
9. 원격지 이름을 origin 으로 설정 디폴트 이름 사용
10. 원격지 url 붙여넣기
11. url에 토큰 정보 추가하기
12. push 을 클릭하여 정상적으로 동작하는지 확인
